### AI Prompting Strategy

# 🎯 Objective

Use AI as assistant, not replacement.

# 🧠 Prompt Rules

- Be specific:
  - ❌ “Fix this”
  - ✅ “Fix JWT unauthorized error in Flask route”

- Provide context:
  - File name
  - Error logs
  - Expected output

# 🔄 Iteration Approach

- Ask for solution
- Test manually
- Refine prompt if needed

# ⚠️ Constraints

- Do not blindly copy AI code
- Always test backend APIs
- Validate frontend integration

# ✅ Coding Standards

- Clean naming
- Modular structure
- Error handling
- Comments where needed
