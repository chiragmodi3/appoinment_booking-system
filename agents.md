### AI Agent Guidelines

# 🎯 Goal

Build a scalable full-stack appointment booking system.

# 📏 Rules

- Follow MVC structure
- Do not mix business logic in routes
- Use reusable services
- Keep APIs RESTful
- Maintain clean code

# 🔐 Security Rules

- Always validate input
- Use JWT for protected routes
- Do not expose sensitive data

# 🎨 UI Rules

- Use Tailwind CSS
- Maintain consistent design
- Use responsive layouts

# ❌ Avoid

- Hardcoding values
- Duplicate logic
- Direct DB calls in frontend
